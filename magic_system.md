# 魔法体系（感情→魔力）

**担当：3本柱の「感情魔法」**

この世界の魔法は、感情を“魔力”へ変換することで発動する。  
感情の強度・持続・複雑さ・混ざり方が、魔力量・出力・制御難度に影響する。

## 1. 魔法の原理
- 魔法は「感情」を **魔力に変換** することで発動する。
- 感情の強度、持続、複雑さ、混ざり方が、魔力量・出力・制御難度に影響する。

## 2. 感情タイプと適性
- 感情の種類（例：怒り、悲しみ、喜び、恐怖、愛着、羞恥…）によって、得意な魔法系統が変化する。
- 魔法は感情の“種類”で形が変わるのではなく、同じ魔法の質（効率・安定性・上限・反動）が変わる

## 3. 固有魔法（個人差の根拠）
- 個人の **感情履歴**（どんな体験をし、どんな感情を積み重ねてきたか）が、固有魔法の種になる。
- 固有魔法は「その人の物語」を反映し、同じ感情タイプでも結果は一致しない。

## 4. 実運用の観点（世界のリアリティ用）
- 訓練：感情制御・呼吸法・儀式・詠唱・契約更新など、魔法の再現性を上げる文化が発展する。
- 社会：感情を“資源”として扱う倫理が生まれやすく、教育・宗教・法制度へ影響する。
