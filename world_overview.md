# 世界観概要（LIVIA / リヴィア）

## 1. コアコンセプト
- この世界には、一般公開していない中核設定（コアコンセプト）がある。
- 公開版ドキュメントでは、ネタバレ防止のため中核設定の詳細を伏せる。
- 魔法は「感情」をエネルギー源とし、社会・文化・歴史・個人の生き方に深く影響する。

> 重要：設定追加は公開ルールと、非公開の中核設定の両方に矛盾しないこと。

## 2. この世界で“感情”が持つ意味
- 感情は主観的な経験であると同時に、物理現象に干渉できるエネルギー（魔力）へ変換されうる。
- 個人の感情履歴は「才能」「適性」「固有魔法」へ影響し、人生の軌跡そのものが力になる。

## 3. 物語の大きな到達点（メタ構造）
- 物語の最終到達点は非公開（ネタバレ防止）。
- 公開版では、各時代のテーマと感情魔法の発展のみを扱う。
