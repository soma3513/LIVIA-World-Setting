# LIVIA / リヴィア 世界設定リポジトリ（ドラフト）

**感情が、魔法になる世界。**

LIVIA（リヴィア）は、感情を“魔力”へ変換できる中世風ファンタジー世界です。  
怒り・喜び・悲しみ・恐れ——感情の質や積み重ねは、魔法の得意分野や **固有魔法** として現れます。  
人は精霊と契約し、魔法と共に生きる。けれど魔法は電気と強く干渉し、電気文明は発展しにくい。  
そのため世界は中世の姿を保ったまま、**魔法技術だけが局所的に高度化**していきます。

このリポジトリは、ぴっぴさんが構築中の中世風ファンタジー世界「LIVIA（リヴィア）」の設定を、共同制作しやすい形で整理するための土台です。

- 不足している箇所は **TODO** として明示しています（後から追記しやすいように）。
- 共同制作フロー（提案 → レビュー → 採用）については `contribution_workflow.md` を参照してください。
- 中核に関わるネタバレ設定は、方針に従って **非公開で管理**します（詳細は `ACCESS_POLICY.md`）。

## 公開版の核となる3要素（3本柱）
このリポジトリでは、公開資料として主に次の3要素を中心に設定を組み立てます。

- **感情魔法**：感情が魔力へ変換され、体験の積み重ねが固有魔法として現れる
- **精霊契約**：精霊との契約によって魔法の扱い方・社会構造・価値観が形作られる
- **文明制約（中世維持）**：魔法と電気が干渉するため電気文明が発展しにくく、世界は中世風を保つ

> ※以下は公開資料に直書きしません：世界の根本に関わるネタバレ（シミュレーションの事実／観測者の正体と目的／終着点の詳細など）
