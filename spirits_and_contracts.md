# 精霊と契約

**担当：3本柱の「精霊契約」**

LIVIAでは、精霊との契約が魔法運用の基盤になる。  
契約の前提（対価・義務・価値観）が、文化や倫理、社会構造を形作っていく。

## 1. 精霊の正体
- 精霊は **感情の化身** である。
- 感情が集積する場所・出来事・人の内面などから発生/具現化しうる。

## 2. 契約の役割
- 精霊と **契約** することで、魔法の使用が可能になったり、出力が安定したりする。
- 契約は相互関係であり、一方的な支配ではない（※この方針は必要に応じて明文化）。

## 3. 共生モデル（ミトコンドリア比喩）
- 精霊は、人間に対して「ミトコンドリア的」な共生関係を持つイメージ。
- 例：人が持つ感情を変換し、魔力として利用可能にする“共生器官”のような役割。
