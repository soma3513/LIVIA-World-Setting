# 記述スタイルガイド（用語・表記）

## 1. 固有名詞
- 世界名：LIVIA（リヴィア）
- 中核設定の固有名詞：非公開管理（公開版には直書きしない）

## 2. コア用語（定義）
- 感情：主観経験／魔力へ変換可能なエネルギー源
- 魔力：感情変換によって得られる力（物理現象に干渉）
- 精霊：感情の化身／契約対象
- 契約：精霊との相互関係の取り決め

## 3. 文体
- 設定資料は「断定→理由→影響」の順に書く
- 追加設定は、最低でも以下のどれかに触れる：
  - 魔法体系への影響
  - 精霊・契約への影響
  - 文明・技術への影響
  - 歴史・政治への影響

## 4. TODOの扱い
- 未決定は「TODO:」で明示し、断定しない
- 後から決める前提の“余白”は歓迎（共同制作向き）
